頻出単語調査アプリ

リンク先 https://com-word-app-ver-2.herokuapp.com/
概要 2000文字以内の英文を左のフォームに入力すると，右のグラフに頻出単語上位10個まで表示される．また，棒グラフをクリックすると wikipedia のページに飛ぶ
使用ツール Python, Django(REST Framework), Vue.js, docker-compose, NLTK, Wikipedia API
制作期間 2021年6月〜現在
git hub https://github.com/mayukorin/com_word_app_ver_2
工夫した点 頻出単語を表示させる時に，正規化やノイズ除去も行った点．また，頻出単語を表示させるだけではなく，その単語の wikipedia にすぐアクセスできるようにした点
苦労した点 単語の wikipedia のページと棒グラフをリンクさせる部分．chart.js の onClick プロパティを使って実現
課題 一般名詞が頻出単語として表示されることが多いからなのか，論文の中の「重要な単語」を抽出できているとは言えない気がする．固有名詞限定にするか，「Note that ~」など，重要そうな接続詞が登場する文章に出てくる単語を調べてもよいのかも
制作理由 自分が研究室で論文を読んでいて，頻出単語（大事な単語）をすぐチェックできるアプリがあったらいいなと思ったから．
Python で開発した理由 頻出単語を調べるのに，正規化・ノイズ除去などが簡単にできる NLTK を使いたいと思っていて，NLTK は Python のパッケージであったから．
Django で開発した理由 Django は，管理画面が自動生成されるため，データの管理が簡単だから．ノイズ除去の対象にする単語を，管理画面を通して簡単に追加できる．
Django REST Framework で開発した理由 頻出単語の棒グラフを表示させる時にchart.js を使っていたため，Django の値を JavaScript に簡単に受け渡ししたかったから．
Vue で開発した理由 Vuetify を使って，手軽に綺麗な web ページを作り立ったから．
